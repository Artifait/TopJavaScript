Правила именования веток

Название ветки строится по следующему правилу: Js{тип задания}{номер}. Если задание состоит из нескольких задач, добавляется Task{номер задачи}.

Примеры:
  • JsPracticeOne
  • JsPracticeOneTaskOne
  • JsModuleOneTaskOne

Где лежат домашки?

Все домашние задания находятся в отдельных ветках данного репозитория. Ветка master содержит общие стили и базовую документацию. Чтобы переключиться на ветку с домашним заданием, используйте команду:

git checkout имя-ветки

Например: git checkout JsPracticeOne
